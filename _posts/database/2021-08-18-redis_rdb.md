## 工具
### rdbtools的安装
rdbtools有三个主要的功能
- 分析静态rdb文件并生成csv格式的内存报告
- 将rdb文件转储成为json格式
- 利用diff工具比较两个rdb文件的不同

通过python的pip安装:
```
./pip install rdbtools
./pip install python-lzf
```
